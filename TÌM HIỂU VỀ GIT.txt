
1. Git là gì?
Git là một hệ thống quản lý phiên bản (Version Control System - VCS) được thiết kế để theo dõi sự thay đổi trong mã nguồn của dự án phần mềm và giữ cho lịch sử làm việc có thể được quản lý hiệu quả. 

2. Cách sử dụng Git:
Để sử dụng Git, bạn có thể thực hiện các bước sau:

Tạo một repository: `git init` để khởi tạo một kho lưu trữ mới.

Thêm file : `git add` để đánh dấu sự thay đổi trong các file.

Lưu sự thay đổi: `git commit` để lưu sự thay đổi với một thông điệp mô tả.

Tương tác với các nhánh (branches): Git cho phép bạn làm việc với nhiều nhánh song song, sử dụng `git branch` và `git checkout`.
